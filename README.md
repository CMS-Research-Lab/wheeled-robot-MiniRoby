## Descripción del proyecto

MiniRoby es un robot móvil diferencial, de bajo costo e impreso en 3D, diseñado para:

- Enseñanza de robótica móvil  
-  Investigación en control y seguimiento de trayectorias  
-  Desarrollo de sistemas embebidos modulares  

El sistema integra:
-  **Arduino Nano** → Control de bajo nivel (motores, PWM, encoders)  
-  **Jetson Nano** → Procesamiento, control y generación de trayectorias  
-  **Estructura impresa en 3D** → Diseño modular y replicable  

---

## 📦 Contenido del repositorio

- 📁 `/stl` → Archivos STL para impresión 3D  
- 📁 `/arduino` → Código para control de motores  
- 📁 `/jetson` → Scripts en Python para control y trayectorias  
- 📁 `/videos` → Ejecución de trayectorias (lineal, circular, sinusoidal)
